# Landing placeholders

Assets temporales para la landing.

## Reemplazos recomendados

- landing-hero-shot.svg
  - Reemplazar por: landing-hero-shot.webp
  - Resolucion sugerida: 1920x1200
  - Peso sugerido: <= 350KB

- landing-hero-video-poster.svg
  - Reemplazar por: landing-hero-video-poster.webp
  - Mismo encuadre del video

- landing-hero-loop.mp4
  - Crear este archivo (actualmente no incluido)
  - Duracion sugerida: 6-10 segundos
  - FPS sugerido: 24
  - Sin audio
  - Peso sugerido: <= 2MB
  - Codec sugerido: H.264
